# ESP32_FreeRtos
ESP32 Arduino: Creando una tarea FreeRTOS

El objetivo de esta publicación es explicar cómo iniciar tareas con las funciones de FreeRTOS.

Introducción
El objetivo de esta publicación es explicar cómo iniciar tareas con las funciones de FreeRTOS . Dado que esto introducirá algunos conceptos complejos, comenzaremos por un ejemplo muy simple en el que crearemos dos tareas que imprimirán algunos mensajes de "Hola mundo" y luego los eliminarán.
Para la mayoría de las funcionalidades que vamos a utilizar en este tutorial, puede verificar el archivo .h aquí . Por favor, consulte esta publicación anterior para una introducción sobre FreeRTOS y tareas.
